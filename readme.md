# Print JSON Sizes CLI

Print the size of JSON keys, in Byte/KB/MB.

`print-json-size ./file.json`
```

     "bin"                  11% 43 b
     "name"                 10% 39 b
     "files"                10% 39 b
     "version"               5% 20 b
     "main"                  5% 20 b

```
